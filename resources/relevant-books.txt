Here are some books that you might find helpful when developing a secure key management system:

    "Cryptography Engineering: Design Principles and Practical Applications" by Niels Ferguson, Bruce Schneier, and Tadayoshi Kohno. This book provides a comprehensive introduction to cryptography and its applications, with a focus on the design and implementation of secure systems.

    "Cryptography for Developers" by Tom Arnold. This book provides a hands-on guide to cryptography for developers, covering topics such as encryption algorithms, hash functions, digital signatures, and key management.

    "Mastering Cryptography and Network Security" by Claudio Lazar and Junaid Ahmed. This book provides a comprehensive overview of cryptography and network security, covering topics such as encryption algorithms, key management, digital signatures, and secure communication protocols.

    "Applied Cryptography: Protocols, Algorithms, and Source Code in C" by Bruce Schneier. This book provides a detailed guide to cryptography, with a focus on the practical aspects of implementing encryption algorithms and secure communication protocols.

    "Security Engineering: A Guide to Building Dependable Distributed Systems" by Ross Anderson. This book provides an overview of security engineering, covering topics such as cryptography, secure key management, and secure software development.

These books can provide you with a solid foundation in cryptography and secure key management, as well as practical guidance for developing secure systems. Additionally, you may also find it helpful to consult online resources and industry standards for best practices and guidance on implementing secure key management systems.
